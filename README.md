# Resume - Vinícius Temóteo Ferrari

This is my personal resume website built with Jekyll and hosted on GitHub Pages.

## Setup

1. Install Ruby and Bundler if you haven't already:
   ```bash
   # On macOS (using Homebrew)
   brew install ruby
   
   # On Windows, download from https://www.ruby-lang.org/en/downloads/
   ```

2. Install Jekyll and dependencies:
   ```bash
   bundle install
   ```

3. Run Jekyll locally:
   ```bash
   bundle exec jekyll serve
   ```

4. Open your browser and navigate to `http://localhost:4000`

## Deployment to GitHub Pages

1. Push this repository to GitHub
2. Go to your repository settings
3. Navigate to "Pages" section
4. Select the branch (usually `main` or `master`) and folder (`/ (root)`)
5. Your resume will be available at `https://yourusername.github.io/repository-name/`

## Customization

- Edit `_data/resume.yml` to update your resume information
- Modify `assets/css/style.css` to change the styling
- Update `_layouts/resume.html` to change the layout structure
